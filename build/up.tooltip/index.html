Tooltips

Elements that have an up-tooltip attribute will show the attribute value in a tooltip when a user hovers over the element.

Unobtrusive behavior
Elements matching these CSS selectors will automatically gain useful behavior when they are added to the DOM tree.
[up-tooltip]

Displays a tooltip when hovering the mouse over this element:

<a href="/decks" up-tooltip="Show all decks">Decks</a>

You can also make an existing title attribute appear as a tooltip:

<a href="/decks" title="Show all decks" up-tooltip>Decks</a>
Javascript methods
Use these methods to programmatically interact with Up.js from your own Javascript code.
up.tooltip.open(elementOrSelector, [options])

Opens a tooltip over the given element.

up.tooltip.open('.help', {
  html: 'Enter multiple words or phrases'
});
Parameters
elementOrSelector Element or jQuery or String
[options.html] String optional

The HTML to display in the tooltip.

[options.position='top'] String optional

The position of the tooltip. Known values are top and bottom.

[options.animation] String optional

The animation to use when opening the tooltip.

up.tooltip.close(options)

Closes a currently shown tooltip. Does nothing if no tooltip is currently shown.

Parameters
options Object

See options for up.animate.
