Utility functions

Up.js comes with a number of utility functions that might save you from loading something like Underscore.js.

Javascript functions
up.util.parseUrl() experimental

Parses the given URL into components such as hostname and path.

up.util.selectorForElement() experimental

Returns a CSS selector that matches the given element as good as possible.

up.util.extend(target, sources...)

Merge the contents of two or more objects together into the first object.

up.util.each(array, block)

Calls the given function for each element (and, optional, index) of the given array.

up.util.map(array, block)

Translate all items in an array to new array of items.

up.util.times(count, block)

Calls the given function for the given number of times.

up.util.isNull(object)

Returns whether the given argument is null.

up.util.isUndefined(object)

Returns whether the given argument is undefined.

up.util.isDefined(object)

Returns whether the given argument is not undefined.

up.util.isMissing(object)

Returns whether the given argument is either undefined or null.

up.util.isGiven(object)

Returns whether the given argument is neither undefined nor null.

up.util.isBlank(object)

Return whether the given argument is considered to be blank.

up.util.presence(object, [tester])

Returns the given argument if the argument is present, otherwise returns undefined.

up.util.isPresent(object)

Returns whether the given argument is not blank.

up.util.isFunction(object)

Returns whether the given argument is a function.

up.util.isString(object)

Returns whether the given argument is a string.

up.util.isNumber(object)

Returns whether the given argument is a number.

up.util.isHash(object)

Returns whether the given argument is an object, but not a function.

up.util.isObject(object)

Returns whether the given argument is an object.

up.util.isElement(object)

Returns whether the given argument is a DOM element.

up.util.isJQuery(object)

Returns whether the given argument is a jQuery collection.

up.util.isPromise(object)

Returns whether the given argument is an object with a then method.

up.util.isDeferred(object)

Returns whether the given argument is an object with then and resolve methods.

up.util.isArray(object)

Returns whether the given argument is an array.

up.util.isDeferred(object)

Converts the given array-like argument into an array.

up.util.copy(object)

Shallow-copies the given array or object into a new array or object.

up.util.merge(sources...)

Creates a new object by merging together the properties from the given objects.

up.util.detect(array, tester)

Passes each element in the given array to the given function. Returns the first element for which the function returns a truthy value.

up.util.any(array, tester) experimental

Returns whether the given function returns a truthy value for any element in the given array.

up.util.compact(array)

Returns all elements from the given array that are neither null or undefined.

up.util.uniq(array)

Returns the given array without duplicates.

up.util.select(array)

Returns all elements from the given array that return a truthy value when passed to the given function.

up.util.nextFrame(block)

Schedules the given function to be called in the next Javascript execution frame.

up.util.once(fun) experimental

Modifies the given function so it only runs once. Subsequent calls will return the previous return value.

up.util.contains(arrayOrString, elementOrSubstring)

Returns whether the given array or string contains the given element or substring.

up.util.only(object, keys...)

Returns a copy of the given object that only contains the given properties.

up.util.resolvedDeferred()

Returns a Deferred object that is already resolved.

up.util.resolvedPromise()

Returns a promise that is already resolved.

up.util.unresolvablePromise() experimental

Returns a promise that will never be resolved.

up.util.remove(array, element)

Removes the given element from the given array.
