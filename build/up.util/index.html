Utility functions

All methods in this module are for internal use by the Up.js framework and will frequently change between releases.

If you use them in your own code, you will get hurt.

Javascript methods
Use these methods to programmatically interact with Up.js from your own Javascript code.
up.util.normalizeUrl([options]) protected

Normalizes URLs, relative paths and absolute paths to a full URL that can be checked for equality with other normalized URL.

By default hashes are ignored, search queries are included.

Parameters
[options.hash=false] Boolean optional
[options.search=true] Boolean optional
up.util.normalizeMethod() protected
up.util.option(args...)

Returns the first argument that is considered present. If an argument is a function, it is called and the value is checked for presence.

This function is useful when you have multiple option sources and the value can be boolean. In that case you cannot change the sources with a || operator (since that doesn't short-circuit at false).

Parameters
args... Array
up.util.cssAnimate(elementOrSelector, lastFrame, [options])

Animates the given element's CSS properties using CSS transitions.

If the element is already being animated, the previous animation will instantly jump to its last frame before the new animation begins.

To improve performance, the element will be forced into compositing for the duration of the animation.

Parameters
elementOrSelector Element or jQuery or String

The element to animate.

lastFrame Object

The CSS properties that should be transitioned to.

[options.duration=300] Number optional

The duration of the animation, in milliseconds.

[options.delay=0] Number optional

The delay before the animation starts, in milliseconds.

[options.easing='ease'] String optional

The timing function that controls the animation's acceleration. See W3C documentation for a list of pre-defined timing functions.

Return value

A promise for the animation's end.

up.util.finishCssAnimate(elementOrSelector) protected

Completes the animation for the given element by jumping to the last frame instantly. All callbacks chained to the original animation's promise will be called.

Does nothing if the given element is not currently animating.

Also see up.motion.finish.

Parameters
elementOrSelector Element or jQuery or String
