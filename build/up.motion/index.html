Javascript methods
Use these methods to programmatically interact with Up.js from your own Javascript code.
up.animate(elementOrSelector, animation, [options])

Animates an element.

The following animations are pre-registered:

  • fade-in
  • fade-out
  • move-to-top
  • move-from-top
  • move-to-bottom
  • move-from-bottom
  • move-to-left
  • move-from-left
  • move-to-right
  • move-from-right
  • none
Parameters
elementOrSelector Element or jQuery or String
animation String or Function or Object
[options.duration] Number optional
[options.easing] String optional
[options.delay] Number optional
Return value
Promise

A promise for the animation's end.

up.morph(source, target, transitionOrName, [options])

Performs a transition between two elements.

The following transitions are pre-registered:

  • cross-fade
  • move-top
  • move-bottom
  • move-left
  • move-right
  • none

You can also compose a transition from two animation names separated by a slash character (/):

  • move-to-bottom/fade-in
  • move-to-left/move-from-top
Parameters
source Element or jQuery or String
target Element or jQuery or String
transitionOrName Function or String
[options.duration] Number optional
[options.easing] String optional
[options.delay] Number optional
Return value
Promise

A promise for the transition's end.

up.transition(name, transition)

Defines a named transition.

Parameters
name String
transition Function
up.animation(name, animation)

Defines a named animation.

Parameters
name String
animation Function
up.motion.none()

Returns a no-op animation or transition which has no visual effects and completes instantly.

Return value
Promise

A resolved promise
