Viewport scrolling

This modules contains functions to scroll the viewport and reveal contained elements.

By default Up.js will always scroll to an element before updating it.

Javascript methods
Use these methods to programmatically interact with Up.js from your own Javascript code.
up.viewport.defaults([options])
Parameters
[options.duration] Number optional
[options.easing] String optional
[options.padding] Number optional
[options.view] String or Element or jQuery optional
up.scroll(viewOrSelector, scrollPos, [options]) protected
Parameters
viewOrSelector String or Element or jQuery
scrollPos Number
[options.duration] String optional
[options.easing] String optional
Return value
Deferred
up.reveal(element, [options]) protected
Parameters
element String or Element or jQuery
[options.view] String or Element or jQuery optional

Return value
