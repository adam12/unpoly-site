Module up.tooltip
up.tooltip.config = config

Sets default options for future tooltips.

Value
[config.position] String optional

The default position of tooltips relative to the element. Can be either "top" or "bottom".

[config.openAnimation='fade-in'] String optional

The animation used to open a tooltip.

[config.closeAnimation='fade-out'] String optional

The animation used to close a tooltip.
