Pop-up overlays

Instead of linking to another page fragment, you can also choose to "roll up" any target CSS selector in a popup overlay. Popup overlays close themselves if the user clicks somewhere outside the popup area.

For modal dialogs see up.modal instead.

Unobtrusive behavior
Elements matching these CSS selectors will automatically gain useful behavior when they are added to the DOM tree.
a[up-popup]

Opens the target of this link in a popup overlay:

<a href="/decks" up-modal=".deck_list">Switch deck</a>

If the up-sticky attribute is set, the dialog does not auto-close if a page fragment below the popup overlay updates:

<a href="/decks" up-popup=".deck_list">Switch deck</a>
<a href="/settings" up-popup=".options" up-sticky>Settings</a>
Parameters
[up-sticky] optional
[up-position] optional
[up-close]

When an element with this attribute is clicked, a currently open popup is closed.

Javascript methods
Use these methods to programmatically interact with Up.js from your own Javascript code.
up.popup.defaults(options)
Parameters
options.animation String
options.position String
up.popup.open(elementOrSelector, [options])

Opens a popup overlay.

Parameters
elementOrSelector Element or jQuery or String
[options.url] String optional
[options.position='bottom-right'] String optional
[options.animation] String optional

The animation to use when opening the popup.

[options.duration] Number optional

The duration of the animation. See up.animate.

[options.delay] Number optional

The delay before the animation starts. See up.animate.

[options.easing] String optional

The timing function that controls the animation's acceleration. up.animate.

[options.sticky=false] Boolean optional

If set to true, the popup remains open even if the page changes in the background.

[options.history=false] Object optional
up.popup.source()

Returns the source URL for the fragment displayed in the current popup overlay, or undefined if no popup is open.

Return value
String

the source URL

up.popup.close(options)

Closes a currently opened popup overlay. Does nothing if no popup is currently open.

Parameters
options Object

See options for up.animate.
